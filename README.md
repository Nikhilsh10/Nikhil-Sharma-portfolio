# Nikhil Sharma Portfolio

**Nikhil Sharma** – Machine Learning Engineer & Data Analyst

Welcome to my personal portfolio website showcasing AI/ML projects, data science analyses, and full‑stack web development work. Built with **Next.js 16**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **shadcn/ui**, featuring glass‑morphism, dark‑mode, and micro‑animations.

## Features
- Premium single‑page layout with smooth scrolling
- Dark‑mode first, glass‑morphism navbar
- Project detail modal, testimonials carousel, blog section
- Resend‑powered contact form (server‑less email)
- SEO ready: Open Graph, structured data, sitemap, robots.txt
- CI workflow with GitHub Actions

## Projects
- **AgriFuture‑INDIA** – Crop recommendation system using ML.
- **RAG‑QA‑System** – Retrieval‑augmented generation for question answering.
- **Deepfake Detector** – CNN‑based deepfake detection.
- **Bank Loan Case Study** – Data analysis and predictive modeling.
- **Other AI/ML experiments** – Various notebooks and demos.

## Getting Started
```bash
git clone https://github.com/Nikhilsh10/Nikhil-Sharma-portfolio.git
cd Nikhil-Sharma-portfolio
npm install
npm run dev
```

## Deployment
Push to GitHub and connect the repository to Vercel (or Netlify). Set the following environment variables in the hosting dashboard:
- `RESEND_API_KEY`
- `RESEND_FROM_EMAIL`
- `RESEND_TO_EMAIL`
